# Online Retail Sales Performance Analysis

This project was completed as part of the Future Interns Data Science & Analytics Internship Program.

## Project Overview
The project focuses on analyzing online retail sales data using Power BI to identify trends, customer behavior, top-performing products, and revenue patterns.

## Tools Used
- Power BI
- Power Query
- Microsoft Excel

## Key Features
- Interactive dashboard
- Revenue analysis
- Product performance analysis
- Country-wise sales analysis
- Monthly sales trends
- Customer insights

## Data Cleaning
Data cleaning was performed in Power Query by:
- Removing duplicates
- Handling missing values
- Formatting data types
- Creating calculated columns

## Insights
- Sales increased toward the end of the year
- A few products generated most of the revenue
- Some products sold in large quantities but produced lower revenue
- Sales were concentrated in specific countries

## Recommendations
- Increase marketing during low-performing months
- Promote high-revenue products
- Expand into strong-performing markets
- Review pricing strategies
